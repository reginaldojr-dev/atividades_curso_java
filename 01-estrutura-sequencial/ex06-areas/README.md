# Ex06 — Áreas

**Entrada:** três doubles `A B C`  
**Saída:** áreas de triângulo, círculo, trapézio, quadrado e retângulo (3 casas)

### Executar
```bash
mvn -q clean compile
mvn -q exec:java
```

### Exemplo
Entrada:
```
3.0 4.0 5.2
```
Saída:
```
TRIANGULO: 7.800
CIRCULO: 84.949
TRAPEZIO: 18.200
QUADRADO: 16.000
RETANGULO: 12.000
```
